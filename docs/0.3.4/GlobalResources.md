### `LoadContent<T>(path, cm)`

Loads a file (Cross-platform)

| Parameter | Datatype  | Purpose |
|-----------|-----------|---------|
|`path` |`string` |Path to file |
|`cm` |`ContentManager` |Content manager to use |
